P3Web is unable to locate the published file that you requested.

There are several potential reasons why you may be seeing this message:

  • The file is in DRAFT status (check the "Version" information above). DRAFT documents do not publish to P3Web.
  • If this is the result of a document status change, the Paradigm 3 Publisher may not have had adequate time to publish the file. The standard setting is 15 minutes
  • The document version may have been changed again since this Action Item was triggered.
  • The file may have failed to publish. Please ask a Paradigm 3 Administrator to check the Publisher Log to confirm.

If this message persists, please contact your Paradigm 3 Administrator to report the issue.
